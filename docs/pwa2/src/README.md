## How to use

### Install

:::: code-group

::: code-group-item yarn

```bash
yarn add -D vuepress-plugin-pwa2@next
```

:::

::: code-group-item npm

```bash
npm i -D vuepress-plugin-pwa2@next
```

:::

::::

### Usage

:::: code-group

::: code-group-item TS

```ts
// .vuepress/config.ts
import { pwa } from "vuepress-plugin-pwa2";

export default {
  plugins: [
    pwa({
      // your options
    }),
  ],
};
```

:::

::: code-group-item JS

```js
// .vuepress/config.js
const { pwa } = require("vuepress-plugin-pwa2");

module.exports = {
  plugins: [
    pwa({
      // your options
    }),
  ],
};
```
